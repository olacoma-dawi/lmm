# Aportaciones a la algorítmica 
**Algoritmo para encontrar el máximo de un conjunto**. El algoritmo puede ser escrito en lenguaje C++ cómo vemos en el siguiente código: 
~~~
int max(int c[], int n) 
{ 
 int i, m = c[0]; 
 for (i = 1; i < n; i++) 
 if (c[i] > m) m = c[i]; 
 return m; 
} 
~~~
